# Data League Project - No Show Prediction

## Project Description

This project aims to predict whether a patient will miss their appointment using machine learning models. The goal is to improve prediction performance based on given tabular data.

## Models Used

* CatBoost
* LightGBM
* XGBoost
* Ensemble Learning

## Pipeline

* Data loading and preprocessing
* Feature engineering
* Model training
* Evaluation using AUC score
* Final prediction generation

## Result

Final AUC Score: 0.50738

## Project Structure

* main.py: all training and prediction code
* requirements.txt: required libraries
* CSV files: dataset used for training and testing

## Notes

* Dataset is not included due to size.
* Developed as part of the Data League '26 competition.
